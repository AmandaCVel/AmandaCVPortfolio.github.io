Steps For This Project:

* Data Cleaning- going through the data verify there are no duplicates, verify the data is spelled correctly, fixing minor and major mistakes, double check for blanks.
* Data Processing- will apply some formulas and build some customized columns in the data, which will be used for the visualization.
* Data Analysis- using the aggregation data to build different PivotTables (AKA PTs) and build charts out of them.
* Data Visualization- building custom charts, use custom formatting, and then place them in the dashboard.
* Dashboard- the final output, which will be built based on the 4-5 steps as stated from above. Will be dynamic to gain insight on the below requirements: the data output using different filters and slicers.

Skills Used in This Project:

Tableau- Line Chart, Bar Chart, Creating Hierarchy(ies), Aggregatted Measures,
Data Cleaning- going through the data verify there are no duplicates, verify the data is spelled correctly, fixing minor and major mistakes, double check for blanks and deleting or dismissing them.

Data Analysis- using the aggregation data to build different PivotTables (AKA PTs) and build charts out of them.
Data Visualization- building custom charts, use custom formatting, and then place them in the dashboard; Timeline, Filters.

Client wants to create an Executive Overview Dashboard for the years 2014 through December 2017 to gain insights on the below requirements:

Primary KPIs- Sales, Profits, Profit Ratio, Profit per Order, Sales per Customer Avg. Discount, and Quantity.
Secondary KPI's- Monthly Sales by Category
Secondary KPI's- Monthly Sales by Segment
Interactive Map Showing all the states in the US that the company sells to.
Interactive Filters Being able to select the Region of the US, the Order Date, and two different ways view the Profit Ratio.

These KPIs can be found directly above the map. The KPIs included in this project are the "Sales" Total, "Profit" Total, "Profit Ratio" Total. "Profit per Order" Total, "Sales per Customer" Total, "Avg. Discount" Total, and "Quantity" Total for the duration of Janurary 3, 2014 - December 30, 2017.

This map is interactive to change the specific state or to view multiple states at once. This allows users to easily access necessary information at just a click! This map can be found in the middle of the dashboard.

These interactive filters allows finding necessary information easier. These filters include "Region," "Order Date," "Profit Ratio," and "Profit Ratio." The Region Filter will divide the US into different Regions Such as "Central" showing the centered states, "East" showing the most Northern Eastern states, followed by "South," and "West" for Southern and Western states. The Order Date Filter goes back to Janurary 3, 2014 - December 30, 2017. Finally the Profit Ratio Filters, the first goes all the way to -21.7% - 37.0% Profit Ratios, the second goes from -50% - 50% Profit Ratios.

This chart shows the Monthly Sales by Category. The three categories include: Furniture, Office Supplies, and Technology.

This chart is similar to the previous one, however, this chart includes the Monthly Sales by Segment. These Segments include Consumer, Corporate, and Home Office.
