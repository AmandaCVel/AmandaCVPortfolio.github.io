What Was Done
in Project:

  1. Data Cleaning- going through the data verify there are no duplicates, verify the data is spelled correctly, fixing minor and major mistakes, double check for blanks.
  2. Data Processing- will apply some formulas and build some customized columns in the data, which will be used for the visualization.
  3. Data Analysis- using the aggregation data to build different PivotTables (AKA PTs) and build charts out of them.
  4. Data Visualization- building custom charts, use custom formatting, and then place them in the dashboard.
  5. Dashboard- the final output, which will be built based on the 4-5 steps as stated from above. Will be dynamic so that they can see the data output using different filters and slicers.
