Interactive Project:
Coffee Sales

  • View Project Online

Steps For This Project:

I) Requirement: Gathering from client.
II) Identify the Stakeholders of the Project.
III) Data Cleaning as per the Requirements.
IV) Data Processing by adding some customized columns in the data/tables.
V) Data Analysis by Pivot Tables and Excel Functions.
VI) Data Visualization to create charts and custom sheets to show the insights.
VII) Report/Dashboard creation from start to end.

Stakeholders- particular peoples, organizations, or the departments that will be using this dashboard for their use:

* Owner/Founder
* CEO
* CFO
* Store Manager
* Accountant
* Cost Estimators

Skills Used in This Project:

Excel- Pivot Tables, Line Chart, Bar Chart,
Data Cleaning- going through the data verify there are no duplicates, verify the data is spelled correctly, fixing minor and major mistakes, double check for blanks and deleting or dismissing them.
Data Processing- will apply some formulas and build some customized columns in the data, which will be used for the visualization.
Data Analysis- using the aggregation data to build different PivotTables (AKA PTs) and build charts out of them.
Data Visualization- building custom charts, use custom formatting, and then place them in the dashboard.

Client wants to create a Road Accident Dashboard for the years 2019 and 2022 to gain insight on the below requirements:

* Primary KPI- Total Sales
* Secondary KPI's- Sales by Country
* Secondary KPI's- Top 5 Customers
* Timeline showing Janurary 2019 - August 2022.
* Roast Type Name
* Loyalty Card Holders
* Size of bag of coffee beans by kilograms.
